Kansas City Crime - Predictive Analysis

This project used SciKit Learn machine learning and Tableau to analyze current and potential future trends for crime in Kansas City, Missouri. The predictions were made using two different models (linear and polynomial) to contrast and compare effective modeling techniques. All data is from data.kcmo.gov. For a more in-depth look at the process, visit our GitHub
