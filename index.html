Kansas City Crime - Predictive Analysis

Crime in Kansas City has become a national discussion involving cricial topics in social justice, resource access, and weapon laws. In order to make effective change, one must first understand the trends of the data. This project used SciKit Learn machine learning and Tableau to analyze current and potential future trends for crime in Kansas City, Missouri. The predictions were made using two different models (linear and polynomial) to contrast and compare effective modeling techniques. All data is from data.kcmo.org. For a more in-depth look at the process, visit our GitHub

The Tableau Kansas City crime prediction above uses the data obtained by the polynomial machine learning model of this project. The zip code comparison of total reported crime provides a clear indication that the areas in Kansas City with the highest reported crimes lies East of 71 Highway.





Crime Forecast

The Tableau Kansas City crime forecast allows for the comparison of machine learning models on the data, to validate or invalidate the predictions created by our own models. This forecast shows the same trend of decreasing reported crime throughout Kansas City.
