Kansas City Crime - Predictive Analysis

This project used SciKit Learn linear regression machine learning and Tableau to analyze current and potential future trends for crime in Kansas City, Missouri. All data is from data.kcmo.gov. For a more in-depth look at the process, visit our GitHub

Has crime in Kansas City increased since 2014? What trajectory are we headed in for the future? Look through the data collections found on this landing page.
